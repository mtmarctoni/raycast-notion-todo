## Getting Started

1. **Clone the repo:**

   ```sh
   git clone https://github.com/your-username/raycast-notion-todo.git
   cd raycast-notion-todo
   ```

2. **Install dependencies:**

   ```sh
   npm install
   ```

3. **Set up your Notion integration:**
   - Create a Notion integration and share your databases with it.
   - Add your Notion tokens and database IDs in the Raycast extension preferences.

4. **Build the extension:**

   ```sh
   npm run build
   ```

5. **Load into Raycast:**
   - Follow Raycast‚Äôs instructions to load your extension.

## Keyboard Shortcuts

- **Cmd+Shift+Up/Down:** Cycle workspace filter
- **Cmd+Shift+T:** Move todo to tomorrow
- **Cmd+Shift+M:** Move todo to next Monday
- **Cmd+E:** Edit todo
- **Cmd+Return:** Mark as completed
